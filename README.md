# 十二时辰养生助手 - Chrome 插件

根据中国传统中医十二时辰理论，提供当前时辰养生建议和完整对照表。

## ✨ 功能特性

- 🕐 **当前时辰** — 自动识别当前时辰，实时显示养生要点
- 📋 **对照表** — 十二时辰完整对照，一目了然
- 📖 **详细建议** — 点击任意时辰查看饮食、运动、情志、禁忌详细建议
- ⏱️ **实时时钟** — 显示当前时间，精确到秒
- 🎨 **精美界面** — 深色国风主题，金色太极图标

## 🚀 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**
4. 选择本项目文件夹
